# FAQ Chatbot

A simple FAQ chatbot that uses NLP techniques to match user questions with predefined FAQs using cosine similarity.

## Features

- Natural language processing with NLTK for text preprocessing
- TF-IDF vectorization and cosine similarity for question matching
- RESTful API with Flask backend
- Modern, responsive chat interface
- Confidence scores for matches
- Suggested questions from FAQ list
- Cross-Origin Resource Sharing (CORS) enabled
